ใส่ Web App URL ที่ได้จากการ Deploy Google Apps Script (ดูคู่มือด้านล่าง)

📖 คู่มือ Setup (ทำครั้งเดียว)

  1. เปิด Google Sheets → สร้าง Spreadsheet ใหม่ → คัดลอก Sheet ID จาก URL
    https://docs.google.com/spreadsheets/d/THIS_IS_THE_ID/edit
  2. ใน Spreadsheet → Extensions → Apps Script
  3. ลบโค้ดเดิมทั้งหมด → วางโค้ดจากไฟล์ apps_script.gs ที่ให้มา
  4. แก้ YOUR_GOOGLE_SHEET_ID ให้เป็น Sheet ID ของคุณ
  5. คลิก Deploy → New deployment → Web App
  6. Execute as: Me / Who has access: Anyone → Deploy
  7. คัดลอก Web App URL → วางในช่องด้านบน → กดบันทึก
🌐 วิธี Deploy บน GitHub Pages (รับ URL ฟรี)
  1. สมัคร GitHub (ฟรี)
  2. สร้าง Repository ใหม่ เช่น chemical-stock
  3. อัปโหลดไฟล์ index.html นี้เข้า Repository
  4. ไปที่ Settings → Pages → Branch: main → Save
  5. ได้ URL: https://username.github.io/chemical-stock/
  6. แชร์ URL นี้ให้พนักงานทุกคน — เปิดได้ทั้งมือถือและคอมพิวเตอร์
